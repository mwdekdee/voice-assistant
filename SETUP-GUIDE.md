## 📝 ขั้นตอนการติดตั้ง:

### 1. ลบ Repository เก่า (ถ้ามี):

1. ไปที่: https://github.com/mwdekdee/Ass
2. กด **3 จุด `...`** → **Settings**
3. เลื่อนลงล่างสุด → หัวข้อ **"Danger Zone"** (สีแดง)
4. กด **"Delete this repository"**
5. พิมพ์: `mwdekdee/Ass`
6. กด **"I understand, delete this repository"**

---

### 2. สร้าง Repository ใหม่:

1. ไปที่: **https://github.com/new**
2. ตั้งชื่อ: **`voice-assistant`**
3. เลือก **Public**
4. ✅ **Add a README file**
5. กด **"Create repository"**

---

### 3. อัปโหลดไฟล์:

1. กด **"Add file"** → **"Upload files"**
2. **เปลี่ยนชื่อไฟล์** ก่อนอัปโหลด:
   - voice-assistant-pwa-FINAL.html → `voice-assistant-pwa.html`
   - manifest-FINAL.json → `manifest.json`
   - sw-FINAL.js → `sw.js`
   - icon-generator-FINAL.html → `icon-generator.html`
   - README-FINAL.md → `README.md`
3. **อัปโหลดไฟล์ทั้ง 5** พร้อมกัน
4. กด **"Commit changes"**

---

### 4. เปิด GitHub Pages:

1. กด **Settings** (แถบบน)
2. เลือก **Pages** (เมนูซ้าย)
3. **Source**: Deploy from a branch
4. **Branch**: main → / (root)
5. กด **Save**
6. **รอ 2-3 นาที**

## 📱 ติดตั้งบน iPhone:

1. **เปิด Safari** → ไปที่ลิงก์ข้างบน
2. จะเห็น**ปุ่มไมค์สีฟ้า** 🎤
3. กดปุ่ม **แชร์** (กลางล่าง)
4. เลื่อนลงหา **"Add to Home Screen"**
5. ตั้งชื่อ: **เลขาส่วนตัว**
6. กด **เพิ่ม**
7. เสร็จ! มีไอคอนบน Home Screen แล้ว 🎉

---

## 🔔 การใช้งานครั้งแรก:

1. เปิดแอปจาก Home Screen
2. จะมีกล่องเหลือง **"ขออนุญาตแจ้งเตือน"**
3. กด **"อนุญาต"** → จะมี popup → กด **"อนุญาต"** อีกครั้ง
4. จะมีแจ้งเตือนทดสอบ **"พร้อมแจ้งเตือนคุณแล้ว!"**

---

## 🧪 ทดสอบ:

1. กดปุ่มไมค์ → พูด **"ทดสอบ"**
2. ตั้งเวลา **2 นาที**ข้างหน้า
3. เลือกหมวด **"ส่วนตัว"**
4. กด **"บันทึก"**
5. **ปิดแอป** หรือล็อคหน้าจอ
6. รอ 2 นาที → จะมีแจ้งเตือน! 🔔

---
