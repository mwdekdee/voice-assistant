# 🎙️ เลขาส่วนตัว - Voice Assistant PWA

แอปเลขาส่วนตัวที่ใช้เสียงพูดสั่งงาน ตั้งเวลา และรับแจ้งเตือนแม้ไม่ได้เปิดแอป

## ✨ ฟีเจอร์หลัก

### 🎤 ควบคุมด้วยเสียง
- พูดสั่งงานภาษาไทยได้เลย
- ไม่ต้องพิมพ์
- รองรับภาษาไทยและอังกฤษ

### ⏰ แจ้งเตือนอัจฉริยะ
- **แจ้งเตือนแม้ปิดแอป** (ต้องติดตั้งเป็น PWA)
- แจ้งเตือนแม้หลับหน้าจอ
- เสียงดัง + สั่น
- กดเลื่อนหรือทำเครื่องหมายเสร็จได้

### 📋 จัดการงานง่าย
- หมวดหมู่: งาน, ส่วนตัว, ธุรกิจ, เร่งด่วน
- ตั้งเวลาเตือนล่วงหน้า (5 นาที - 1 วัน)
- Swipe เพื่อทำเครื่องหมายเสร็จ
- ข้อมูลบันทึกถาวร

### 📲 ติดตั้งได้เหมือน App จริง
- ไอคอนบน Home Screen
- ทำงานแบบ Native App
- ไม่ต้องผ่าน App Store/Play Store

## 🚀 วิธีติดตั้ง

### สำหรับมือถือ Android:

1. เปิดไฟล์ `voice-assistant-pwa.html` ใน Chrome
2. จะมีแบนเนอร์ขึ้นด้านบน "ติดตั้งเป็นแอป"
3. กด **"ติดตั้ง"**
4. แอปจะปรากฏบน Home Screen
5. กด **"อนุญาต"** เมื่อขออนุญาตแจ้งเตือน

### สำหรับ iPhone (iOS):

1. เปิดไฟล์ `voice-assistant-pwa.html` ใน Safari
2. กดปุ่ม **แชร์** (ไอคอนสี่เหลี่ยมมีลูกศรชี้ขึ้น)
3. เลื่อนลงหา **"Add to Home Screen"**
4. ตั้งชื่อแล้วกด **"เพิ่ม"**
5. แอปจะปรากฏบน Home Screen

### สำหรับคอมพิวเตอร์:

1. เปิดใน Chrome, Edge, หรือ Brave
2. ดูที่ Address Bar จะมีไอคอน ⊕ หรือ 🖥️
3. กดเพื่อติดตั้ง
4. แอปจะเปิดในหน้าต่างแยก

## 📖 วิธีใช้งาน

### เพิ่มงานใหม่:

1. **กดปุ่มไมค์สีฟ้า** กลางหน้าจอ
2. **พูด** เช่น "ส่งเอกสารให้ลูกค้า" หรือ "โทรหาคุณแม่"
3. **ตั้งค่า**:
   - วันที่และเวลา
   - หมวดหมู่
   - เตือนล่วงหน้า
4. กด **"บันทึก"**

### จัดการงาน:

- **กดที่งาน** → ทำเครื่องหมายเสร็จ/ยกเลิก
- **กดถังขยะ** → ลบงาน

### เมื่อถึงเวลาเตือน:

- แจ้งเตือนจะปรากฏขึ้น + เสียง + สั่น
- เลือก:
  - **"เสร็จแล้ว"** → ทำเครื่องหมายงานเสร็จ
  - **"เลื่อน 15 นาที"** → เลื่อนการแจ้งเตือน

```

icon-512.png               ← ไอคอนขนาด 512x512
icon-generator.html        ← สำหรับสร้างไอคอน (ถ้าต้องการ)
README.md                  ← ไฟล์นี้
```

## 🔧 ข้อกำหนดเทคนิค

- **เบราว์เซอร์**: Chrome, Edge, Safari, Brave (เวอร์ชันล่าสุด)
- **ระบบปฏิบัติการ**: Android 5+, iOS 11.3+, Windows 10+, macOS 10.13+
- **การแจ้งเตือน**: ต้องอนุญาต Notification Permission
- **ไมโครโฟน**: ต้องอนุญาต Microphone Permission

## ⚠️ ข้อควรระวัง

1. **ต้องติดตั้งเป็น PWA** ถึงจะแจ้งเตือนได้แม้ไม่เปิดแอป
2. **iOS มีข้อจำกัด**: การแจ้งเตือนบน iPhone จะทำงานได้ดีขึ้นใน iOS 16.4+
3. **ประหยัดแบตเตอรี่**: ปิดการแจ้งเตือนที่ไม่จำเป็น
4. **ข้อมูลเก็บในเครื่อง**: ข้อมูลจะอยู่ในเบราว์เซอร์ ถ้าลบแคช = ข้อมูลหาย

## 🎯 ข้อดีเทียบกับ App ตลาด

✅ **ฟรี 100%** - ไม่มีค่าใช้จ่าย ไม่มี Premium  
✅ **ไม่ต้องสมัครสมาชิก** - ใช้ได้เลยทันที  
✅ **ไม่เก็บข้อมูลส่วนตัว** - ข้อมูลอยู่ในเครื่องคุณเท่านั้น  
✅ **หน้าตาง่าย** - ไม่ซับซ้อน ไม่งง  
✅ **เสียงภาษาไทยแม่นยำ** - ใช้ Web Speech API ของ Google  
✅ **ไม่มีโฆษณา** - สะอาด ใช้งานโฟกัส  

## 🛠️ การปรับแต่ง

### เปลี่ยนสี Theme:

แก้ไขไฟล์ `manifest.json`:
```json
"theme_color": "#4dabf7",  // เปลี่ยนเป็นสีที่ต้องการ
"background_color": "#ffffff"
```

### เพิ่มหมวดหมู่:

แก้ไขไฟล์ `voice-assistant-pwa.html` ในส่วน:
```html
<select id="categoryInput">
  <option value="work">งาน</option>
  <option value="personal">ส่วนตัว</option>
  <option value="business">ธุรกิจ</option>
  <option value="urgent">เร่งด่วน</option>
  <!-- เพิ่มของคุณเองที่นี่ -->
</select>
```

## 🐛 แก้ปัญหา

### ไม่มีเสียงแจ้งเตือน:
- ตรวจสอบว่าเสียงเครื่องไม่เงียบ
- ตรวจสอบว่าอนุญาต Notification แล้ว

### ไมโครโฟนไม่ทำงาน:
- ตรวจสอบว่าอนุญาต Microphone Permission
- ลองรีเฟรชหน้าเว็บ
- ลองเปิดใน Chrome/Safari

### ไม่มีปุ่มติดตั้ง:
- ต้องเปิดผ่าน HTTPS หรือ localhost
- ลองเปิดในเบราว์เซอร์อื่น
- iOS: ต้องใช้ Safari เท่านั้น

### ข้อมูลหาย:
- อย่าลบ Browser Cache
- อย่าใช้โหมด Private/Incognito
- สำรองข้อมูลเป็นประจำ (Export feature อาจเพิ่มในอนาคต)
